## ⚙️ 技术栈 (Technology Stack)

我们基于高性能、模块化的架构进行开发，确保系统稳定与可扩展性。

| 类别 | 技术 | 描述 |
| :--- | :--- | :--- |
| **后端框架** | 自定义PHP框架 | 轻量、灵活，专注于高性能数据处理。 |
| **前端技术** | Tailwind CSS, JavaScript | 快速构建现代化、响应式 UI 界面。 |
| **数据库** | MySQL | 可靠的关系型数据库支持。 |
| **架构模式** | MVC (Model-View-Controller) | 清晰分离业务逻辑、数据和界面。 |
| **设计理念** | 模块化设计 | 前后端组件化，功能模块高度分离。 |

## 🔑 主要功能概览

### 💻 前台功能 (Frontend Features)

* ✅ **视频中心：** 视频列表、搜索、播放、分类导航。
* ✅ **用户认证：** 完整的登录与注册流程。

### 👑 后台功能 (Admin Features)

* 📊 **管理中心：** 仪表盘统计、系统设置、主题管理。
* 📦 **内容管理：** 视频管理、评论管理。
* 👥 **用户管理：** 直播管理、用户权限管理。

### 🔧 安装程序功能 (Installation Flow)

* 🧭 **向导式安装：** 环境检测、数据库配置、管理员设置。
* 🛡️ **安全提示：** 安装完成后强制提示删除安装目录。

## 🏗️ 项目结构 (Project Structure)

项目采用清晰的 MVC 模块化结构，便于开发者理解和扩展。
```
├── app/
│   ├── Core/              # 核心框架
│   │   ├── App.php        # 应用入口
│   │   ├── Router.php     # 路由系统
│   │   ├── Controller.php # 控制器基类
│   │   ├── Model.php      # 模型基类
│   │   ├── Database.php   # 数据库连接
│   │   └── Auth.php       # 认证系统
│   ├── Modules/           # 功能模块
│   │   ├── Frontend/      # 前端模块
│   │   ├── Admin/         # 后台管理模块
│   │   ├── Video/         # 视频模块
│   │   ├── User/          # 用户模块
│   │   ├── Live/          # 直播模块
│   │   └── Comment/       # 评论模块
│   ├── Languages/         # 语言包
│   └── Helpers/           # 辅助函数
├── config/                # 配置文件
├── database/              # 数据库文件
│   ├── migrations/        # 迁移文件
│   └── seeds/             # 种子数据
├── public/                # 公共目录
│   ├── assets/            # 静态资源
│   │   ├── css/           # 样式文件
│   │   ├── js/            # JavaScript文件
│   │   ├── images/        # 图片
│   │   └── vendor/        # 第三方库
│   ├── uploads/           # 上传文件
│   └── themes/            # 主题目录
│       └── default/       # 默认主题
│           ├── css/       # 主题样式
│           ├── js/        # 主题脚本
│           ├── images/    # 主题图片
│           ├── layouts/   # 主题布局
│           └── views/     # 主题视图
├── storage/               # 存储目录
├── tests/                 # 测试文件
├── autoloader.php         # 自动加载器
├── index.php              # 项目入口
└── project_structure.md   # 项目结构文档
```
