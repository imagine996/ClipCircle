## 技术栈

- **后端框架**: 自定义PHP框架
- **前端技术**: Tailwind CSS, JavaScript
- **数据库**: MySQL
- **架构模式**: MVC (Model-View-Controller)
- **模块化设计**: 功能模块分离，前后端分离

## 安装使用

1. **配置环境**
   - PHP 7.4+
   - MySQL 5.7+
   - Web服务器（Apache/Nginx）

2. **安装向导**
   - 运行目录/public
   - 输入：http://你的域名/install


3. **运行项目**
   - 将项目部署到Web服务器
   - 访问项目根目录

## 开发指南

### 路由配置
路由配置在 `app/Core/Router.php` 中进行设置。

### 创建控制器
在对应模块的 `Controllers` 目录下创建控制器类，继承 `App\Core\Controller`。

### 创建视图
在对应模块的 `Views` 目录下创建视图文件。

### 创建模型
在对应模块的 `Models` 目录下创建模型类，继承 `App\Core\Model`。

### 主题开发
1. **创建主题目录**
   在 `public/themes/` 目录下创建主题文件夹，如 `mytheme`

2. **主题结构**
   ```
   mytheme/
   ├── css/
   │   └── theme.css       # 主题样式文件
   ├── js/
   │   └── theme.js        # 主题JavaScript文件
   ├── images/             # 主题图片目录
   ├── layouts/            # 主题布局目录
   │   └── default.php     # 默认布局文件
   └── views/              # 主题视图目录
       └── home/           # 首页视图目录
           └── index.php   # 首页视图文件
   ```

3. **配置主题**
   在 `config/theme.php` 中添加主题信息

4. **切换主题**
   在后台管理界面的「主题管理」页面切换主题

### 主题系统特点
- 所有主题文件统一存放在 `public/themes` 目录下
- 支持热切换主题，无需重启服务
- 主题包含完整的布局、视图、样式和脚本
- 可以基于默认主题进行定制

## 注意事项

- 项目使用了模块化设计，功能模块之间保持低耦合
- 前端使用了响应式设计，适配不同设备
- 后台管理系统使用了侧边栏导航和深色主题
